## 第 33 章　总结

本项目完整复现了 Bertsekas (2019) 的鲁棒最短路(RSP)算法体系,并以实验实证其理论。主要工作与结论如下:

### 33.1 理论复现

系统梳理了论文的建模与理论:RSP 的 minimax 建模(对手取最坏后继)、proper/improper 策略与 `J_μ=最长路`、三算子 `H/T_μ/T`、以及把 RSP 嵌入**半压缩抽象 DP** 后得到的核心结论——在假设 2.1 下,`J*` 是 Bellman 算子 `T` 在 `R(X)` 内的**唯一不动点**,最优策略必 **proper**,且 `T^k J → J*`(Prop 4.3)。我们对 Prop 4.1、4.2、4.3 给出了完整证明,并对三个核心算法(VI 有限终止、Dijkstra-like 永久标号与 N+1 终止、PI 单调收敛)给出了**逐行完整证明**(定理 A/B/C)。

### 33.2 算法与工程复现

实现了 VI、PI、Dijkstra-like 三个核心求解器,以及 rollout、三种 deterministic baseline、exhaustive ground-truth,全部通过统一接口对接。每段代码都与论文公式一一对应(附录 A)。工程上做到:零告警(`-Wall -Wextra -Wpedantic`,可 `-Werror` 强制)、ASan/UBSan 干净、ctest 3/3、CI 多平台矩阵 + 冒烟。

### 33.3 数据与实验

用 layered DAG **构造性保证假设 2.1/5.1**,并用陷阱图族验证失败模式。四组实验逐层印证理论:
- **实验 1**:四算法在 toy 图一致(`J*=[7,1,101,4,100,0]`),鲁棒策略 worst-cost 7 vs 名义 102;
- **实验 3**:三算法 `avg_value` 一致(↔`J*` 唯一)、dijkstra 迭代=N(↔N+1 终止)、vi 有限终止;
- **实验 4 / 4b**:VI worst-cost 在所有 `s≥2` 最低,baseline 在陷阱图上失效率 85%(↔Prop 4.1),配对统计 VI 100/100 不劣,多种子稳健(900/900,`66.97±0.75`);
- **复现保真度**:**~28 万张随机图 vi=pi=dijkstra=exhaustive 零不匹配**,是 `J*` 唯一不动点(Prop 4.3a)的最强实证。

### 33.4 改进贡献

在复现之外,本项目还做了系统的工程与方法学加固(exhaustive 规模守卫、IO 校验、graceful failure 统一、CSV 转义、CI `-Werror`/多平台、随机图跨算法一致性测试、陷阱图族、配对/多种子统计),并指出了算法层面(堆优化、Gauss-Seidel、乐观 PI、rollout)与理论层面(随机扩展、负权、不完美信息)的进一步方向。

## 第 34 章　不足与展望

### 34.1 不足

1. **数据族单一**:当前聚焦非负边权的 layered DAG(假设 4.3),未覆盖论文的负环(假设 4.1)与零环 + 扰动法(Prop 4.5)情形;
2. **Dijkstra-like 为 naive 扫描实现**(`O(N²·A·S)`),未做堆优化,故在 n=200 上未体现其理论复杂度优势;
3. **规模有限**:实验止于 `n=200`,未触及大规模(需先做堆优化);
4. **未实现随机扩展**(式 6.1)与不完美状态信息的 RSP;
5. **运行时间为亚毫秒级**,效率结论对测量噪声敏感,需更多重复 + 计时框架以给出严谨的 runtime 置信区间。

### 34.2 展望

- **算法**:堆优化 Dijkstra-like、Gauss-Seidel VI、乐观/异步 PI,使效率结论更具说服力并扩展到大规模;
- **数据**:补负环/零环图族,完整覆盖论文三类假设;引入真实地图(追逃)数据;
- **理论**:实现并实验随机扩展(RSP+SSP 混合,式 6.1),探索不完美状态信息归约;
- **统计**:bootstrap 置信区间 + 配对符号检验,给出更严谨的方法学。

总之,本项目不仅"复现了算法",更做到了"忠实于理论(28 万张图零不匹配)、严谨于工程(零告警 + sanitizer + CI)、扎实于实验(四组 + 陷阱 + 配对 + 多种子)",并为后续研究留出了清晰的优化与延伸路径。
